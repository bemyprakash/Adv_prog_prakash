# Foodizz: Advanced Food Delivery API

## The Core Concept
Foodizz is a comprehensive backend infrastructure for a modern food delivery application. The goal is to build an ecosystem that seamlessly integrates real-time interactions across multiple stakeholders—customers, restaurant owners, delivery agents, and customer support.

## High-Level Features
- **Role-Based Access:** Distinct experiences for 5 different types of users using secure JWT authentication.
- **Robust Order Lifecycle:** Tracking an order completely from a shopping cart generation to final delivery.
- **Dedicated Resolution Center:** A built-in ticketing system to handle edge cases like failed payments, cold food, or delivery agent emergencies.
- **Enterprise Object-Oriented Design:** Leveraging strict OOP principles (Inheritance, Polymorphism) via SQLAlchemy to map complex real-world relationships cleanly into a database.

## Academic Objective
This project serves as a capstone demonstration of Advanced Programming techniques, highlighting clean API routing, strict data validation schemas, and database encapsulation to present a fully production-ready backend backbone.
